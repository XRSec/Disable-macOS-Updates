## 📦 安装步骤：

1️⃣ 双击 "安装屏蔽更新" 文件

2️⃣ 输入您的管理员密码（密码输入时可见）

3️⃣ 按回车键确认运行

4️⃣ 等待安装完成

---

## 🗑️ 卸载步骤：

1️⃣ 双击 "卸载屏蔽更新" 文件

2️⃣ 输入您的管理员密码（密码输入时可见）

3️⃣ 按回车键确认运行

4️⃣ 等待卸载完成
